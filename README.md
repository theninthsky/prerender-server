This is a _[Prerender](https://github.com/prerender/prerender)_ server example.

A deployed version is available at: https://prerender-server.onrender.com/render?url={WEBSITE_URL}

For example: _[https://prerender-server.onrender.com/render?url=https://client-side-rendering.pages.dev/lorem-ipsum](https://reqbin.com/qgordjg0)_

_Please note that since this is running on a shared CPU in the free **[Render](https://render.com)** plan, responses will take a long time to return (some might even fail)._
